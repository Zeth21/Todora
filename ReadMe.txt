**Todora**

Todora is a full backend project that provides multi-user task management with features for sharing tasks and organizing task processes.

## Technologies

- ASP.NET Core Web API (.NET 9)
- Onion Architecture
- CQRS + MediatR
- Entity Framework Core
- AutoMapper
- JWT Authentication
